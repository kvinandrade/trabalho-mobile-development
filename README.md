# Loja DisKO - Aplicativo Mobile

Aplicativo mobile desenvolvido em React Native como parte da disciplina de Mobile Development. O projeto simula um sistema simples de gerenciamento de produtos de uma loja de discos de vinil chamada **DisKO**, permitindo visualizar, adicionar, editar e remover produtos.

## Tecnologias Utilizadas

- React Native
- Expo
- TypeScript
- JavaScript
- API REST
- Node.js
- Expo Go

## Funcionalidades do Aplicativo

O aplicativo possui as seguintes funcionalidades:

- Tela de login de usuário
- Listagem de discos cadastrados
- Adicionar novos produtos
- Editar produtos existentes
- Remover produtos
- Navegação entre telas

## Como Executar o Projeto

Siga os passos abaixo para rodar o projeto localmente.

### 1. Clonar o repositório

git clone https://github.com/kvinandrade/trabalho-mobile-development.git

### 2. Entrar na pasta do projeto

cd trabalho-mobile-development

### 3. Instalar as dependências

npm install

### 4. Iniciar o projeto

npx expo start

Após iniciar o projeto, será exibido um QR Code no terminal ou no navegador.

### 5. Executar no celular

1. Instale o aplicativo Expo Go no celular
2. Escaneie o QR Code exibido no terminal
3. O aplicativo será aberto automaticamente
